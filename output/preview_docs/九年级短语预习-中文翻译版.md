| set out | 出发；开始 | Try to set out in the right way. | 试着用正确的方式出发；开始。 |
| separate from | 与……分开 | I learned about separate from in English class. | 我在英语课上学习了与……分开。 |

## 预习方法

1. 每天学习 5 到 10 个短语。
2. 先看中文意思，再读英文例句。
3. 用自己的校园生活再造一个句子。
4. 第二天先复习旧短语，再学新短语。
